Le Jeu

Ce projet est un jeu vidéo de type puzzle / énigme. Dans ce vous incarnez un scientifique qui a pour but de réparer sa machine à voyager dans le temps dans le but d'échapper à vos ravisseurs qui veulent vous empêcher de compléter votre machine, pour ce faire il vous faudra naviguer à travers trois niveaux tout résolvant des énigmes et des puzzles. Pour vous aider dans cette quête vous pourrez vous faire accompagnez par vos amis qui pourront vous rejoindre. Mais ce n'est pas tout, vous aurez également des capacités, comme par exemple celle de revenir dans le temps.
Dans ce jeu vous allez évoluer dans un environement crée dans un style LowPoly. C'est un style qui se veut par essence minimaliste, pour faire cette environement il convient de crée des objets en 3D avec un nombre de polygone limité. Contrairement à un jeu qui se veut réaliste celui-ci exhibe volntairement les fomres carré des modèle pour donner cette impression de simplicité.
