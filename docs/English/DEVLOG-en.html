Devlog

This section of the website is dedicated to Devlogs. It's a section that is very useful to stay updated on the game's news and what we are aiming for in the future of the game.

You can find the developers' progress here between two updates.

09 March 2023

Devlog 1

For this first DevLog, we have the complete skeleton of each theme in the design of our game. This means having a visual representation of each element that composes it. The website has a defined skeleton that we will gradually complete, along with our progress in the Unity project and communication. The Artificial Intelligence, on the other hand, is in its prototype form and only has the basic movement algorithm. Additionally, the menu includes a button to exit the game and another one to exit the application. Lastly, for the networking part, we can host a local game and have multiple instances join it.

Each task has been created for this first milestone in order to have all the necessary foundations to be organized and coordinated within the group for all the upcoming steps. For the next milestone, we will continue building upon the foundations we have established so far.

April 21, 2023

Devlog 2

So far, all the tasks have been progressing well. The AI is complete, the game is starting to be playable, and we have a clear idea of the adopted graphical style.

The game is progressing nicely, and a first version should be available around June.

June 2, 2023

Devlog 3

A first version of the game is now available!! It includes the initial 3 levels. You can now embark on this journey as a brilliant scientist, reconstructing your time-travel machine and venturing through levels to solve various puzzles and mysteries.

You can download the game on the Game page.
